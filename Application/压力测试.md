ab工具

Apache服务器的性能测试工具,它可以测试安装Web服务器每秒处理的HTTP请求.

语法
```shell
ab {选项} {参数}

# 选项
-A：指定连接服务器的基本的认证凭据；
-c：指定一次向服务器发出请求数；
-C：添加cookie；
-g：将测试结果输出为“gnuolot”文件；
-h：显示帮助信息；
-H：为请求追加一个额外的头；
-i：使用“head”请求方式；
-k：激活HTTP中的“keepAlive”特性；
-n：指定测试会话使用的请求数；
-p：指定包含数据的文件；
-q：不显示进度百分比；
-T：使用POST数据时，设置内容类型头；
-v：设置详细模式等级；
-w：以HTML表格方式打印结果；
-x：以表格方式输出时，设置表格的属性；
-X：使用指定的代理服务器发送请求；
-y：以表格方式输出时，设置表格属性。

# 参数
主机：被测试主机。
```

# 评估服务器性能

## 查看服务器参数

[参考](https://www.cnblogs.com/gllegolas/p/11697581.html)

cpu

```
# cpu信息：
cat /etc/cpuinfo
cat /proc/cpuinfo

# 查看cpu物理核数
cat /etc/cpuinfo |grep 'physical id' | sort |uniq|wc -l

# 查看cpu逻辑核数
cat /etc/cpuinfo | grep processor | wc -l

# 查看cpu是几核cpu
cat /proc/cpuinfo |grep 'cores' | uniq

# 查看cpu型号
cat /etc/cpuinfo | grep 'model name'
```

硬盘

```
df -h
```

内存

```
free -h
```
