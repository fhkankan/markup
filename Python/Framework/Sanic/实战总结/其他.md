## 测试程序日志

```python
from sanic.log import logger

# 打印到屏幕：
if __name__ == '__main__':
    from sanic.log import set_logger
    set_logger(filename="", level="INFO")

# 打印到文件
if __name__ == '__main__':
    from sanic.log import set_logger
    set_logger(filename="logs/myfile.log", level="INFO")
```
