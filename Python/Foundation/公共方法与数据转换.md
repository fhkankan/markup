## 运算符

```python
# +
# 合并，适用于：字符串、列表、元组
print("123"+"abc")
print([1, 2, 'a']+['b', True])
print((1, 2, True)+('a', 3))

# *
# 复制，适用于：字符串、列表、元组
print("1"*4)
print([1,2]*2)
print((1,2)*3)

# in
# 元素是否存在，存在返回True,不存在返回False
# 适用于：字符串、列表、元组、字典，集合
if 'a' in "abcde":
    print("yes")
if 'a' in ["a",2]:
    print("yes")
if "a" in ("a","b"):
    print("yes")
# in 判断的是键
if "a" in {'a':1 }:
    print("yes")

# not in
# 元素是否不存在，不存在返回True， 存在返回False
# 适用于：字符串、列表、元组、字典，集合
```

## 切片截取

```python
# 适用于：字符串、列表、元组
# 格式：[起始：结束：步长]
# 区间从"起始"开始，到"结束"位的前一位结束（不包含结束位本身)，步长表示选取间隔

#字符串不可变，列表可变，当进行切片操作时，其实是操作一个副本
# 切片语法：[起始：结束：步长]
# 左闭右开,第一个下标为0，倒数第一个下标-1
# a = 'abcdef'
# a[:3],a[0:3],a[0:3:1]等价
# fedcba
# print(a[-1::-1])
list1 = ['a',3,4,5,6]
print(id(list1))
print(id(list1[:]))
print(id(list1[2:]))
```

# 拷贝

```
浅拷贝：对于一个对象的顶层拷贝，拷贝了引用，并没有拷贝内容

深拷贝：对于一个对象所有层次的拷贝(递归)

拷贝其他方式：
分片表达式可以赋值一个序列(浅)
a = [1, 2]
b = a[:]
字典的copy方法可以拷贝一个字典(浅)
d = dict(name="z", age=27)
c = d.copy()

注意：
copy.copy对于可变类型，会进行浅拷贝
copy.copy对于不可变类型，不会拷贝，仅仅是指向

d = c	将d指向c所指向的空间
d = copy.copy(c) 复制所有c指向的数据到一个新空间，但不会递归拷贝
d=copy.deepcopy(c) 将c指向的空间进行递归拷贝
```

# 转换

## 整数的转换

```python
# float(x) 将x转换为一个浮点数
print(float('5'))
print(type(float('5')))
# 浮点型字符串不能直接转为int类型，需要先转为浮点型
print(type(int(float('10.00'))))

# complex(real[,imag])
# 将 real 和 imag转换到一个复数，实数部分为 real，虚数部分为 imag。real和 imag是数字表达式,imag缺失时默然为0

# chr(x)
# 将一个整数转换为一个Unicode字符

# ord(x)
# 将一个字符转换为它的ASCII整数值

# hex(x)
# 将一个整数转换为十六进制字符串

# oct(x)
# 将一个整数转换为八进制字符串

# bin(x)
# 将一个整数转换为一个二进制字符串

# bool(x)
# 将x转换为布尔类型

# int(x[,base]) 
# 将x转换为一个十进制整数

# long(x,[base])
# 将一个字符转换为long类型，base表示字符的进制

print(float(10.2))
print(complex(3))
print(char(65))
print(ord('A'))
print(bin(5))
print(oct(20))
print(hex(20))
print(int("16"))
print(int("11", 8))
```

## 字符串与表达式

```python
asscii(x)
# 把对象装换为ASCII码表示形式，必要时使用转义符表示特定的字符
ascii('你好')
ascii('a')

bytes(x)
# 用来生成字节串，或把指定对象转换为特定编码的字节串
bytes()  # 生成空的字节串
bytes(3)  # 生成长度为3的字节串
bytes("你好", 'utf-8')  # 把字符串转换为字节串，等价于"你好".encode("utf-8")

str(x)
# 将对象x转换为字符串，给程序员看
# str_num = str(123)
str_num = str('123')
print(type(str_num))
print(str_num)

repr(x)
# 将对象x转换为表达式字符串，给计算机看
b = repr(123)
b = repr('123')
print(type(b))
print(b)

eval(str)
# 用来计算在字符串中的有效表达式，并返回一个对象
a = eval("5")
print(a)
print(type(a))
print(eval("'abc'"))
# 报错
# print(eval('abc'))
#
print(repr(4+3))
print(eval("4+3"))
```
